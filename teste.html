teste de git e github

Como fazer comit e enviar ao github?

para clonar repositório: git clone "link do repositorio"

1. para listar as configurasções do Git:git config --global --list

2. se não for seu usuário e email: configurar usando git config --global user.name "Raquel"

3. nao esquercer na primeira vez ira pedir autorização do git

4. salvar o arquivo que esta escrevendo.

5. comando para criar um arquivo: git add nomeDoArquivo

6. comando para gravar commit:git commit nomeDoArquivo -m "sua mensagem"

7. comando parea enviar ao repositorio github:git push origin main

8. comando para ver atolizaçao: git pull "link do github"

para limpar as credenciais:cmdkey /delete git: https://github.com/vitoriaraay/portifolio.git
